## 快速开始

1. 先安装bable-node
``` bash
npm i -g @babel/core @babel/node

##  命令行输入
babel-node --version
## 输出 7.x版本以上就可以了
7.6.1
```

2. 初始化项目
``` bash
npm install 或者 yarn
```

3. 运行
``` bash
npm start
```
